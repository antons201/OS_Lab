1. Создаем файл с помощью creat, записываем 300КБ с помощью write далее очищаем файл при помощи ftruncate,
закрываем с помощью close и удалям при помощи remove.

2. Реализовали открытье файла, запись в него по биту 300*1000 раз и закрываем файл с помощью POSIX и библиотеки stdio.h
Реализация с помощью библиотеки работает быстрее, т.к. в первом случае программа выполняет команды в операционной системе.

3. Считываем файл, пробегаемся по содержимому и считаем количество строк. Если необходимо вывести больше строк, чем в файле, то выводим всё.
Иначе бежим по тексту, пропускаем лишние строки и выводим нужное количество.

4. Используем стандартную функцию stat() и выводим необходимые поля.

5. Открываем дирректорию, бежим по всем файлам и дирректориям (при помощи opendir и readdir). Если есть флаг, то выводим необходимую доп.информацию.

6. Открываем дирректорию, бежим по всем файлам и дирректориям (при помощи opendir и readdir). В первом случае выводим всё, что нашли.
Во втором случае начинаем выводить дерево, найдя нужную дирректорию. В третьем добавляем проверку на глубину.
